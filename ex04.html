form태그의 중요한 속성은 action과 method가 있다
action : 다른 페이지로 이동할 경로 설정
method : 전송방식 결정. 없으면 기본 GET 방식
-get
- url뒤에 파라미터를 붙여서 데이터를 전송
- 데이터를 전송하는 경우 최대 글자수는 2048
- url에 값이 보이기 때문에 보안에 취약하다고 할 수 있다
-post
- HTTP body에 담아서 데이터를 전송하는 방식
- http에 묶어서 보내기 때문에 대용량의 데이터를 보낼 때 사용
- GET방식보다 보안상 우위에 있다고 표현
